🎮 Interactive Testing

Öffne die Hauptanwendung und teste die Features:

📝 Test-Anweisungen

1. Network Analysis Testing

  1. Lade ein Netzwerk (z.B. "Family Data" oder "Medium Network")
  2. Öffne das GUI-Panel rechts (lil-gui)
  3. Navigiere zu "Analyse" → "Netzwerkanalyse"
  4. Teste folgende Funktionen:
    • 📊 "Netzwerk-Statistiken" - Sollte Popup mit Statistiken zeigen
    • 🎯 Wähle einen Knoten aus und klicke "Knoten-Metriken"
    • 🏆 "Top-Knoten anzeigen" - Zeigt wichtigste Knoten
    • 👥 "Communities erkennen" - Findet Knotengruppen

2. Path Finding Testing

  1. Navigiere zu "Analyse" → "Pfadfindung"
  2. Wähle einen Knoten aus und klicke "Startknoten setzen"
  3. Wähle einen anderen Knoten aus und klicke "Zielknoten setzen"
  4. Klicke "Pfad finden" - Pfad sollte farbig hervorgehoben werden
  5. Teste "Pfad-Info anzeigen" für Details
  6. Teste "Animation umschalten" für bewegte Visualisierung

3. Performance Testing

  1. Lade ein großes Netzwerk ("Large Network" oder "Mega Network")
  2. Navigiere zu "Performance" im GUI
  3. Teste folgende Funktionen:
    • 📈 "Performance-Stats" - Zeigt FPS, Speicher, etc.
    • 💡 "Empfehlungen" - Optimierungsvorschläge
    • ⚙️ "Auto-Optimierung" - Automatische Anpassungen
    • 🔧 Ändere "Max. sichtbare Knoten" und beobachte Effekt

4. Integration Testing

  1. Teste Kombination der Features:
    • Finde Pfad zwischen zentralen Knoten
    • Analysiere Performance bei verschiedenen Netzwerkgrößen
    • Verwende Nachbarschafts-Highlighting mit Pfadfindung

🐛 Bekannte Test-Punkte

Zu überprüfen:
  • Alle GUI-Elemente sind sichtbar und funktional
  • Keine JavaScript-Fehler in der Browser-Konsole
  • Performance-Optimierungen greifen bei großen Netzwerken
  • Pfad-Visualisierung ist klar erkennbar
  • Netzwerk-Statistiken sind plausibel
